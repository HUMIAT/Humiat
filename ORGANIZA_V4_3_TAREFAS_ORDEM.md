# ORGANIZA V4.3 — Tarefas com Ordem Manual

## Ajustes implantados

- Data e hora reais no painel.
- Menu mobile com botão central para criar tarefa.
- Projeto permanece como cadastro administrativo inicial.
- Departamentos/Etapas permanecem dentro do Projeto.
- Tarefa volta a ser o foco principal do uso diário.
- Cada tarefa mostra um número de ordem.
- Cada tarefa pode subir ou descer com setas.
- Exclusão de tarefa respeita a regra: administrador pode excluir qualquer tarefa; usuário comum só exclui tarefa sob sua responsabilidade.
- A ordem é manual, não prioridade.

## Conceito fechado

Projeto e Departamento/Etapa são configurados no início.
No dia a dia, o usuário trabalha em Tarefas.
A ordem da tarefa define o que será feito primeiro.
