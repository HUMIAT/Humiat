# ORGANIZA V4.2 — Projetos / Etapas

## Ajustes realizados

- Tela **Projetos / Etapas** virou a central do fluxo administrativo.
- Botão **Novo projeto** aparece no topo da tela para usuários com permissão.
- O botão **Visualizar** foi substituído por **Abrir fluxo**.
- Ao abrir um projeto, o sistema mostra primeiro os **Departamentos / Etapas**.
- A criação de tarefa ficou dentro da etapa correta, evitando confusão entre etapa e tarefa.
- Menu lateral foi simplificado: removido o item redundante **Criar Projeto**.
- Mobile usa Projetos / Etapas como ponto central de configuração.
- Usuário sem permissão é redirecionado para tela de **Acesso não permitido**.
- Exclusão de tarefa continua respeitando regra: admin pode excluir qualquer uma; usuário comum só exclui tarefa dele.

## Conceito correto

Projeto = fluxo principal.

Departamento / Etapa = fases internas do projeto.

Tarefa = execução dentro de uma etapa.
