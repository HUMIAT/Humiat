# HUMIAT - Site Institucional

Site FastAPI da HUMIAT.

## Rodar local

```powershell
py -m pip install -r requirements.txt
py -m uvicorn app:app --reload
```

Acesse:

```text
http://127.0.0.1:8000
```

## Render

Build Command:

```bash
pip install -r requirements.txt
```

Start Command:

```bash
uvicorn app:app --host 0.0.0.0 --port $PORT
```
