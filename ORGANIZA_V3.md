# Organiza V3

## Ajustes aplicados

- Painel passou a ter seletor de usuário: Junior, Debora e Luiz.
- O painel mostra os números e tarefas do usuário selecionado.
- Todos continuam podendo visualizar todos os usuários.
- Dependência da tarefa foi corrigida para mostrar apenas tarefas do mesmo projeto.
- Departamento / Etapa foi corrigido para mostrar apenas etapas do projeto selecionado.
- O backend também valida se a etapa pertence ao projeto antes de salvar.
- Link para criar tarefa de dependência já leva o projeto e etapa selecionados.
- Layout recebeu ajustes para uso principal em celular.
- Campos de formulário ganharam tamanho melhor para toque.
- Cards, botões e listas foram reorganizados para telas pequenas.

## Regra importante

Uma tarefa só pode depender de outra tarefa do mesmo projeto.
Uma tarefa só pode ser salva em uma etapa pertencente ao projeto selecionado.
